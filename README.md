# Magic Store 🛍️

**Magic Store** is an online marketplace built as a Telegram Mini App. The project is currently in its early stage, with the main focus on project configuration and building the initial UI (pages and layout).

## 🚀 Tech Stack

The project is being developed using the following technologies:

- React  
- TypeScript  
- TON Connect  
- @tma.js SDK  
- Telegram UI  
- Vite  

## 📦 Install Dependencies

After cloning the project, install dependencies using:

```bash
pnpm install
```

## 🤖 Telegram Mini App

This project is designed to run inside Telegram as a Mini App. During development, it can also be launched in a browser.

```bash
pnpm dev:https
```

> Note: Some features only work fully within the Telegram environment.
